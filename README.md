# **README**

## **開発言語**

- Ruby 3.0.1
- Rails 6.1.7

## **就業 Term の技術**

- devise
- お気に入り機能

## **カリキュラム外の技術**

- gem/rails_admin

## **実行手順**

```
$ git clone git@github.sayaka-kamei/POCCHI.git
$ cd POCCHI
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
